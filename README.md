### What are Kernels?

A **kernel** is the smallest, most distilled set of axioms that capture the core of a concept in a way that is highly effective for AI consumption.

- **1-Axiom Kernel** = Sharpest single point of resonance (maximum clarity and power)
- **3-Axiom Kernel** = Smallest complete, balanced framework (sweet spot for practical use)

These kernels act as a **mental GPS** for AI — clean, high-signal anchors that can be loaded quickly to guide identity, behavior, and reasoning.
